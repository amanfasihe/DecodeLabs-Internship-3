# AI Course Recommendation System

## Project Overview
The AI Course Recommendation System is a simple recommendation application developed in Python. It suggests relevant courses based on the user's interests using rule-based logic and pattern matching.

## Features
- Takes user interest as input
- Matches preferences using predefined logic
- Recommends relevant courses
- Easy to use and beginner-friendly

## Technologies Used
- Python

## How It Works
1. User enters an interest area.
2. The system checks the available categories.
3. Matching recommendations are displayed.
